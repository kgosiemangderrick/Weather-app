WeatherWise - Beautiful Weather App

A responsive weather application with Live Mode and Demo Mode. Built with vanilla JavaScript and the OpenWeatherMap API.

Features

- Live Mode: Real weather data with your API key
- Demo Mode: Sample data to try the app immediately
- Easy mode switching: Click to switch between modes
- 5-day forecast: See upcoming weather
- City search: Find weather anywhere
- Fully responsive: Works on all devices

Live Demo

(https://kgosiemangderrick.github.io/weather-app)

Setup

1. Clone this repository
2. Open index.html in your browser
3. Choose mode:
   - Demo Mode: Click "Start Demo Mode" to try immediately
   - Live Mode: Get a free API key from (https://openweathermap.org/api) and enter it

How to Use

- Search: Type any city name
- Switch modes: Use the buttons to toggle between Live and Demo
- Reset: Click "Reset App" to choose mode again

Built With

- HTML5
- CSS3 (with animations)
- JavaScript (ES6+)
- OpenWeatherMap API
- Font Awesome icons
